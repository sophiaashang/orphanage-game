# 🌙 夜访孤儿院

一个像素风格的恐怖推理游戏。

## 故事背景

你是林晓，《城市日报》的调查记者。三天前，你收到一封匿名举报信：
> "向阳福利院的孩子...都消失了..."

今晚，你潜入这座废弃的孤儿院，试图揭开被掩埋的真相。

## 游戏玩法

- **移动**：WASD / 方向键 / 触屏虚拟摇杆
- **交互**：E键 / 点击E按钮 / 触摸调查
- **探索**：在各个房间寻找线索，揭开27条生命的真相
- **多结局**：根据收集的线索，触发不同结局

## 章节结构

1. **第一章：初探孤儿院** - 探索大厅、走廊和院长办公室
2. **第二章：三个孩子** - 发现院长的秘密记录后解锁
3. **第三章：黑暗真相** - 收集3个以上线索后解锁
4. **第四章：终结** - 获得地下室钥匙后解锁

## 技术特点

- 纯前端实现，无需服务器
- 支持PC和移动端
- 自动存档系统
- Web Audio API 音效

## 在线游玩

打开 `index.html` 即可开始游戏。
